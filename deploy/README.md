# Despliegue (Deploy)

Esta carpeta está reservada para futuras configuraciones de despliegue en plataformas como Render o Vercel.

## Futuros Archivos a Incluir
- `render.yaml`: Blueprint de configuración de servicios para Render.
- `vercel.json`: Configuración para Vercel (si se adapta el backend a serverless).
- `Dockerfile`: Para contenerizar la aplicación si es necesario.
- `Procfile`: Comandos de inicio para servicios basados en contenedores (Render/Heroku).
